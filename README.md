# CreditWise - Loan Approval Prediction

## About the Project

This project is based on predicting whether a loan should be approved or not using machine learning. The idea is to use customer data like income, age, and credit history to make predictions.

---

## What I Did

* Cleaned and prepared the dataset using Pandas
* Handled missing values and converted data into usable format
* Performed basic data analysis to understand patterns
* Applied machine learning models like Logistic Regression, KNN, and Naive Bayes
* Compared different models to check which one performs better

---

## Tech Used

Python, Pandas, NumPy, Scikit-learn, Matplotlib, Seaborn

---

## Result

The model was able to predict loan approval with around 80% accuracy. Logistic Regression gave the most stable results.

---

## Why this Project

This project helped me understand how machine learning can be used in real-world problems like banking and risk analysis.
